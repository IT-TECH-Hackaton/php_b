# Инструкция по запуску проекта

## Требования

- Docker Desktop (должен быть запущен и не приостановлен)
- Node.js и npm (для фронтенда)

## Запуск бэкенда (PHP)

1. Убедитесь, что Docker Desktop запущен и не приостановлен

2. Запустите все сервисы:
```bash
cd back_xaxaton2025_php
docker-compose up -d
```

3. Проверьте статус:
```bash
docker-compose ps
```

4. Проверьте логи:
```bash
docker-compose logs app
```

5. Проверьте работу API:
```bash
curl http://localhost:8080/health
```

## Запуск фронтенда

1. Откройте новый терминал и перейдите в папку фронтенда:
```bash
cd frontend-MyAfisha
```

2. Установите зависимости (если еще не установлены):
```bash
npm install
```

3. Запустите dev сервер:
```bash
npm run dev
```

Фронтенд будет доступен по адресу: http://localhost:5173

## Остановка сервисов

Для остановки бэкенда:
```bash
cd back_xaxaton2025_php
docker-compose down
```

Для остановки фронтенда нажмите `Ctrl+C` в терминале где запущен `npm run dev`

## Создание администратора

После запуска бэкенда создайте администратора:
```bash
curl -X POST http://localhost:8080/api/auth/init-admin
```

Или используйте учетные данные по умолчанию:
- Email: admin@system.local
- Password: Admin123!

**Важно:** Измените пароль администратора после первого входа!

## Проверка работы

1. Бэкенд: http://localhost:8080/health - должен вернуть `{"status":"ok"}`
2. Фронтенд: http://localhost:5173 - должен открыться интерфейс приложения

## Переменные окружения

Все переменные окружения настраиваются в файле `.env` или через docker-compose.yml
